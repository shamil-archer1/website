Welcome to Andriod Emulator 2020.1

Thank you for choosing the ANDRIOD-VM Operating System.

Version: 2021.7 | Date of first version release: 30 May, 2020 | Date updated: 20 March, 2021

What's New: Updated information for 2021.7 release.

Author:Shamil Burkhanov

Operating System

  • Current Release Name: ANDRIOD-VM
  • Architecture: 64bit
  • Supported BIOS: Legacy, UEFI & Secure Boot
  • Software Base:UNIX-CORE
  • Desktop Environment: Gnome Lite web interface
  • Media Type:SERVER 20mb at least wifi connection

Preferred Computer Requirements:

  • 1.1 Ghz processor+
  • 1Gb ram+
  • 21MB HDD/SSD+
  • VGA, DVI or HDMI screen capable of 1040x768 resolution+ or larger
  • Windows,Linux or IOS

System Menu


Terminal - Install and Remove software with the Synaptic Package Manager.
Partition Drives - View and edit your hard drive partitions with GParted program.
Printers - Install and set up printers.
System Information - like operating system,device name,browser type,and logs.
System Log - View system your logs, helps with troubleshooting.
System Restore Utility - System restore tool for ANDRIOD 7.1.0 Supports scheduled snapshots, multiple backup levels, and exclude filters. (like System Restore in Windows)
tTerminal - An overview of currently running computer processes. You can kill unresponsive applications here to.
Terminal Emulator - A command line utility like the command prompt in Windows.


Multimedia Menu

Media Player - HTML VIDEO PLAYER, a fantastic all around media player, play mp3's, avi's, mkv's etc. Plays all your media files.
Volume Control - Individually set volume levels for each program that is running and configure your sound devices.
